Do not use DFS platforms (PrizePicks, Underdog Fantasy) for strikeout prop lines or any odds data.

**Why:** PrizePicks had generic lines (4.5 for everyone), which created fake edges and went 0-5 on K props. Underdog lines differ from real sportsbooks. Nick bets on DraftKings so needs DK-specific lines.

**How to apply:** When sourcing odds or prop lines, only use real sportsbooks (DraftKings, FanDuel, BetMGM, Caesars). Currently using BettingPros API with book_id=12 (DraftKings) to get real DK lines with juice.
